Create Links using [link text][link variable] in the post and including[link_var]: http://url.com at the bottom of the .md file. Here’s an example linking to my website :) Also works like this: [link text](https://link.com) ntoledo.me

Adding two spaces after a line of text will continue the paragraph on the next line!

Use {: class="cssClass"} and {: style="color:red"} to add CSS classes and html attributes to a paragraph in markdown
